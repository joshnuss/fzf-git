# fzf-git

**ALPHA SOFTWARE**

`git` powered by `fzf`. Provides autocomplete for paths and branch.

## Supported commands

- `fzf-git add`
- `fzf-git mv`
- `fzf-git rm`
- `fzf-git reset`
- `fzf-git checkout`
- `fzf-git rebase`
- `fzf-git merge`
- `fzf-git show`

All other commands are sent directly to `git`
