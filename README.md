# fzgit

**ALPHA SOFTWARE**

A `git` command alternative powered by `fzf`.

It provides autocomplete for paths and branch.

## Supported commands

- `git add`
- `git mv`
- `git rm`
- `git reset`
- `git checkout`
- `git rebase`
- `git merge`
- `git show`

All other commands are sent directly to `git`
