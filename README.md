# FzF / GIT command

Supercharge `git` with `fzf`. Provides autocomplete for paths and branch.

**ALPHA SOFTWARE**

## Supported commands

- `git add`
- `git mv`
- `git rm`
- `git reset`
- `git checkout`
- `git rebase`
- `git merge`
- `git show`

Any other commands is sent directly to `git`
