# fzgit

**ALPHA SOFTWARE**

`git` powered by `fzf`. Provides autocomplete for paths and branch.

## Supported commands

- `fzgit add`
- `fzgit mv`
- `fzgit rm`
- `fzgit reset`
- `fzgit checkout`
- `fzgit rebase`
- `fzgit merge`
- `fzgit show`

All other commands are sent directly to `git`
